## PHPインストール

1. 対象のバージョンをインストールします。

   - `.\8.2.5.ps1`
   - `.\8.1.18.ps1`
   - `.\8.0.28.ps1`
   - `.\7.4.33.ps1`

2. Pathに[%PHP_HOME%]を追加してください。

    ↑初回のみ
